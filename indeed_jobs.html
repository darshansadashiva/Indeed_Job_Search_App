Role Description

This is a full-time hybrid role for a Junior Web Developer. The Junior Web Developer will be responsible for tasks related to back-end web development, front-end development, web design, programming, and web development. The role is located in Bengaluru and offers flexibility for remote work.

Qualifications

  • Back-End Web Development and Programming skills
  • Front-End Development and Web Design skills
  • Experience in web development
  • Strong problem-solving skills and attention to detail
  • Excellent written and verbal communication skills
  • Ability to work independently and as part of a team
  • Experience with HTML, CSS, JavaScript, and other relevant web development languages and frameworks
  • Knowledge of UI/UX principles and best practices is a plus
  • Bachelors degree in Computer Science, Web Development, or related field
